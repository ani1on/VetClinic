# FastPig — Ветеринарная цифровая платформа

> **FastPig** — современное веб-приложение для ветеринарной клиники, объединяющее запись на приём, каталог товаров и услуг, личный кабинет с карточками питомцев, избранное, отзывы и мощную панель администратора.

## Основные функции

### Для клиентов
- **Авторизация и регистрация** — вход по email/телефону, индикатор сложности пароля
- **Главная лента** — новости, популярные услуги, статистика, отзывы клиентов
- **Каталог товаров** — поиск, фильтры, добавление в корзину/избранное
- **Личный кабинет** — профиль, питомцы, корзина, заказы, отзывы
- **Запись на приём** — выбор питомца, услуги, врача, даты и времени
- **Избранное** — сохранение товаров, услуг и врачей
- **Отзывы** — отправка на модерацию, просмотр и удаление

### Для администратора
- **Дашборд** — метрики записей, пользователей, товаров, отзывов
- **Управление пользователями** — просмотр, смена роли, удаление
- **Управление врачами** — CRUD, привязка к пользователю, фото
- **Управление услугами** — CRUD, категории, цены
- **Управление товарами** — CRUD, склад, изображения
- **Управление новостями** — создание, редактирование, публикация
- **Модерация отзывов** — одобрение/отклонение
- **Просмотр записей** — все визиты, смена статуса
- **Информация о клинике** — контакты, часы работы, соцсети, лицензии

## Локальный запуск

### Требования
- Node.js 18+ и npm
- Python 3.10+ и pip
- Git

### 1. Клонирование

```bash
git clone https://github.com/ani1on/VetClinic.git
cd VetClinic
```

### 2. Бэкенд

```bash
cd back
python -m venv venv
source venv/bin/activate  # Linux/Mac
venv\Scripts\activate     # Windows

pip install -r requirements.txt
```

Без переменной `DATABASE_URL` автоматически используется SQLite. Для PostgreSQL задайте:

```bash
export DATABASE_URL=postgresql+psycopg://user:password@host:5432/dbname
```

Запуск:

```bash
uvicorn main:app --reload --port 8000
```

Документация: http://localhost:8000/docs

### 3. Фронтенд

```bash
cd front
npm install
npm run serve
```

Фронтенд: http://localhost:8080

### 4. Первый запуск

- Администратор создаётся автоматически при первом запуске
- Данные: `admin@fastpig.com` / `Admin123!`
- Для входа в админку измените роль через БД или API

## Изображения

Бэкенд поддерживает URL изображений для врачей (`photo_url`), товаров (`image_url`) и новостей (`image_url`). URL вставляются в админке при создании/редактировании. Изображения отображаются на страницах:

- **Главная** — изображения новостей
- **Каталог** — изображения товаров
- **Избранное** — изображения сохранённых товаров и врачей
- **Админка** — изображения в списках врачей, товаров, новостей
