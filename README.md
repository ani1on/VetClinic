# VetClinic
Название: FastPig
Краткое описание
Сайт ветеринарной клиники представляет собой многофункциональную платформу, созданную для оперативной связи между владельцами животных и медицинским персоналом. Он служит цифровым представительством клиники, предоставляя подробную информацию об услугах, ценах и квалификации врачей. Основная цель ресурса — упростить процесс записи на прием, обеспечить доступ к справочной информации и сформировать доверие к учреждению за счет прозрачности работы и отзывов клиентов.

Целевая аудитория
Аудитория сайта максимально широка и включает всех владельцев домашних и сельскохозяйственных животных. В первую очередь это люди, которым необходима срочная или плановая ветеринарная помощь для своих питомцев (кошек, собак, грызунов, птиц, экзотических животных). Также сайт ориентирован на фермеров и владельцев крупных животных, которым нужны услуги выездной службы, и на ответственных хозяев, интересующихся профилактикой, вакцинацией и грумингом.

Ключевой функционал 
Главная лента (/main): Витрина клиники с акциями, последними новостями из блога, отзывами и краткими карточками основных услуг для быстрой навигации.

Каталог товаров и услуг (/catalog): Детализированный справочник ветеринарных услуг (терапия, хирургия, анализы) и, при наличии, зоотоваров с фильтрацией по цене, специализации врача и направлению лечения.

Авторизация и личный кабинет (/auth, /profile): Разграничение доступа для обычных пользователей и администрации. В личном кабинете пользователя доступен подраздел «Записи» (/profile/appointment) — история и управление визитами, а также подраздел «Избранные» (/profile/favorits) — сохранение понравившихся товаров или услуг для быстрого заказа позже.

Информация о клиннике (/about): Содержит в себе информацию об ветеринарной клиннике и врачах.

Административная панель (/admin): Закрытый раздел для управления контентом (добавление услуг, акций, статей), обработки заявок на запись, просмотра и публикации отзывов, а также ведения учета клиентов и их питомцев.

Описание страниц:
  ГЛАВНАЯ ЛЕНТА:
  Включает в себя меню сайта, по которому будет происходить переключение между страницами, главную ленту с новостями, кнопка для перехода в личный кабинет и кнопка для logout-а.
  КАТАЛОГ ТОВАРОВ:
  Включает поиск по товарам, фильтр по различным критериям, меню страниц, лента товаров, которая зависит от критериев, кнопка для перехода в личный кабинет, переход на страницу записи на услуги, кнопка добавления в избранное.
  ЛИЧНЫЙ КАБИНЕТ:
  Меню страниц, кнопка логаута, кнопка для перехода в избранное, информация клиента.
  ЗАПИСИ:
  меню страниц, кнопка личного кабинета, информация об услуге, ввод личных данных клиента.
  ИЗБРАННОЕ:
  То же самое, то и поиск, только с товарами\услугами клиента, которые были добавлены в избранное.
  ИНФОРМАЦИЯ О КЛИННИКЕ:
  Меню страниц, кнопка лгаута и личного кабинета, информация о клиннике(история линники, достижения, доказательсва квалификации, контактные данные и тп.)
  КАБИНЕТ АДМИНА:
  Меню страниц, логаут, поиск по критериям(поиск пользователя/поиск услуги), возможность работать с пользователями и услугами(удаление/изменение/добавление).

  API запросы:
    /auth/login - авторизация пользователя
    /auth/register - регистрация пользователя
    /user/info - инфо об пользователе
    /user/favorits - избранное пользователя
    /user/records - записи пользователя
    /user/update - изменение данных об пользователе
    /user/add_record - добавление записи
    /user/add_favorits - добавление в избранное
    /user/delete_record - удаление записи
    /user/delete_favorits - удаление избранного
    /service/info - информация об услуге
    /service/create - добавление новой услуги
    /service/delete - удаление услуги
    /service/update - изменение услуги
    /doctor/create - добавление врача
    /doctor/delete - удаление врача
    /doctor/info - инфа б враче
    /doctor/update - изменение врача
