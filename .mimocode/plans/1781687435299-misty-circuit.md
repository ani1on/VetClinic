# План: Подключение Яндекс.Метрики к VetClinic

## Цель
Добавить JavaScript-код Яндекс.Метрики на сайт для отслеживания посещаемости и поведения пользователей.

## Изменения

### 1. `front/public/index.html`
Добавить в `<head>` скрипт Яндекс.Метрики (с плейсхолдером `YOUR_COUNTER_ID`, который нужно заменить на реальный ID из личного кабинета):

```html
<!-- Yandex.Metrika counter -->
<script type="text/javascript">
  (function(m,e,t,r,i,k,a){m[i]=m[i]||function(){(m[i].a=m[i].a||[]).push(arguments)};
  m[i].l=1*new Date();
  for (var j = 0; j < document.scripts.length; j++) {if (document.scripts[j].src === r) { return; }}
  k=e.createElement(t),a=e.getElementsByTagName(t)[0],k.async=1,k.src=r,a.parentNode.insertBefore(k,a)})
  (window, document, "script", "https://mc.yandex.ru/metrika/tag.js", "ym");
  ym(YOUR_COUNTER_ID, "init", {
    clickmap:true, trackLinks:true, accurateTrackBounce:true, webvisor:true
  });
</script>
<noscript><div><img src="https://mc.yandex.ru/watch/YOUR_COUNTER_ID" style="position:absolute; left:-9999px;" alt="" /></div></noscript>
<!-- /Yandex.Metrika counter -->
```

### 2. `front/src/router/index.js`
Добавить `router.afterEach` для отслеживания переходов между страницами (SPA не перезагружает страницу, поэтому метрика не увидит переходы без этого):

```js
router.afterEach((to) => {
  if (typeof window.ym === 'function') {
    window.ym(YOUR_COUNTER_ID, 'hit', to.fullPath);
  }
});
```

## Файлы для изменения
- `front/public/index.html` — добавить скрипт в `<head>`
- `front/src/router/index.js` — добавить отслеживание SPA-переходов

## Верификация
1. Заменить `YOUR_COUNTER_ID` на реальный ID из Яндекс.Метрики
2. `cd front && npm run serve`
3. Открыть сайт, проверить в DevTools: `window.ym` должен быть функцией
4. В личном кабинете Яндекс.Метрики проверить, что посещения фиксируются
